The find method gives the index of one string within another string. Slicing is based on indexes, and there are many use cases where find is used to calculate those indexes.

For example, the following code snippet extracts the text surrounding (1 character before and 1 after) the first "*" character:

text = "ABCDEFG*HIJK*LMN*OPQRSTUVWXYZ"
first_star = text.find("*")
text[first_star-1 : first_star+2]
